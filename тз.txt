Стек технологий: Python (Flask), Flask-SQLAlchemy (ORM), Flask-RESTful (API), PostgreSQL (БД), HTML/CSS/JS (фронтенд).

1. Цель проекта
Разработать веб-приложение для продажи рассады с функционалом интернет-магазина: каталог, корзина, заказы, личный кабинет.

2. Функциональные требования
2.1. Публичная часть (без авторизации)
Главная страница: Список популярных товаров, акции, ссылки на разделы.

Каталог товаров:

Фильтрация по категориям (овощи, цветы, зелень), цене, сорту.

Поиск по названию.

Карточка товара:

Фото, название, описание, цена, наличие.

Кнопка "В корзину".

Корзина:

Добавление/удаление товаров, изменение количества.

Расчет стоимости.

Оформление заказа:

Форма с контактами (имя, телефон, email, адрес доставки).

Выбор способа доставки (самовывоз, почта, курьер).

Оплата (онлайн или при получении).
